1. 수정조건 문제]
테이블이름이 todo_list 의 테이블을 수정하라
필드는 todo_list 의 내용을 '러시아의 전쟁 만행 참혹' 으로 수정하라
조건은 IDX 가 13 인 레코드만 수정하라.

수정전 확인

|  13 | 테스트

명령실행
UPDATE  todo_list   
SET  todo_list = '러시아의 전쟁 만행 참혹'  
WHERE  IDX = 13;


결과
mysql> UPDATE  todo_list   SET  todo_list = '러시아의 전쟁 만행 참혹'  WHERE  IDX = 13;
Query OK, 1 row affected (0.01 sec)
Rows matched: 1  Changed: 1  Warnings: 0




수정후 확인
SELECT * FROM todo_list;

3. 수정조건 문제3]

SET todo_list="국제 국방력 순위 9위"

[3] 조건 => todo_list 의 내용이 "테스트" 가 포함 모든 레코드 수정하라

UPDATE todo_list
SET todo_list="국제 국방력 순위 9위"

// 같은 조건인 경우  % 와일드 카드가 포함 된거는 LIKE 사용

UPDATE todo_list
SET todo_list="국제 국방력 순위 9위"
WHERE todo_list  LIKE  "%테스트%";

수정문제]
테이블이름이  todo_list
조건은 IDX 가 3,5,8번인 레코드 모두를  수정하라
수정필드는 todo_list  수정내용은  "연잎핫도그"


UPDATE todo_list
SET todo_list = "연잎핫도그"  
WHERE IDX=3 OR IDX=5 OR IDX=8;

SELECT * FROM todo_list;


UPDATE todo_list
SET todo_list = "연잎바닐라핫도그"  
WHERE IDX IN (3, 5, 8); // OR 같은 개념
WHERE IDX NOT IN (3, 5, 8); // OR 같은 개념 부정





조건
todo_list 내용이  연잎이 포함되지않은 레코드를 모두 [ NOT LIKE  '%연잎%']
WHERE  todo_list NOT LIKE  '%연잎%'

"우거지 뼈다귀 감자탕" 으로 수정하시오

UPDATE todo_list
SET todo_list = "우거지 뼈다귀 감자탕"  
WHERE  todo_list   NOT  LIKE  '%연잎%';
